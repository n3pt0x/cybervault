# Network

- [Interactsh (CLI)](https://github.com/projectdiscovery/interactsh) | [(web)](https://app.interactsh.com/) - Detecting out-of-band interactions (DNS, TLS, HTTP and much more)
- [webhook.site](https://webhook.site/) - Generates unique URLs, e-mail, DNS.

## VPN

- [Gluetun](https://github.com/qdm12/gluetun) - Lightweight swiss-army-knife-like VPN client to multiple VPN service providers.
- [SSHuttle](https://github.com/sshuttle/sshuttle) - VPN over SSH.

## Tor / I2P

- [i2pd](https://i2pd.website/) - I2P Deamon.
- [NIPE](https://github.com/htrgouvea/nipe) - CLI tool to make Tor your default gateway.

## Sniffing / Spoofing

- [SSLsplit](https://github.com/droe/sslsplit) - Transparent SSL/TLS interception.
- [TCPreplay](https://github.com/appneta/tcpreplay) - Pcap editing and replay tools.
- [Ettercap](https://github.com/Ettercap/ettercap) - MITM attack.
