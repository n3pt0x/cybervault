# 💼 Workbench

## 🧠 Knowledge & Note-Taking

- [AppFlowy](https://github.com/AppFlowy-IO/AppFlowy) - Open-source Notion alternative with AI integration.
- [SiYuan](https://github.com/siyuan-note/siyuan) - Open-source block-based knowledge base with Markdown.
- [Obsidian](https://obsidian.md/) - Powerful knowledge base that works on local Markdown files.
- [Notion](https://www.notion.so/) - All-in-one workspace for notes, projects, and databases.

## 🛠️ Diagramming & Whiteboard

- [Excalidraw](https://excalidraw.com/) - Online whiteboard with hand-drawn feel.
- [draw.io](https://app.diagrams.net/) - JavaScript client-side editor for general diagramming.
- [Mermaid](https://github.com/mermaid-js/mermaid) - Generate diagrams from markdown-like text.

## 🎨 Design & UI

- [Pencil.dev](https://www.pencil.dev/) - Vector design tool.
- [Figma](https://www.figma.com/) - Industry-standard collaborative interface design tool.
- [Penpot](https://penpot.app/) - Open-source alternative to Figma.
- [Coolors](https://coolors.co/) - Color palette generator.

## 📊 Data & Visualization

- [Observable](https://observablehq.com/) - Interactive data visualization with JavaScript.
- [Flourish](https://flourish.studio/) - No-code data visualization for presentations.
- [JSON Crack](https://jsoncrack.com/) - Visualize JSON data as interactive graphs.

## 📚 Documentation & Utilities

- [CodeSandbox](https://codesandbox.io/) - Online IDE for rapid web prototyping.
- [DevDocs](https://devdocs.io/) - Aggregated technical documentation for APIs and frameworks.
- [TableConvert](https://tableconvert.com/) - Convert tables to Markdown, JSON, SQL, and more.
